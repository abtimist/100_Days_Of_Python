This is a project showcasing knowledge in using smtplib and datetime modules to send mails

To run the file:
 - add your email in MY_EMAIL
 - go to your email account setting -> security -> Turn on two step verification
 - Then go to app passwords -> Create an app password -> Copy the password displayed -> paste it in PASSWORD
 - in 'to_addrs' add the recipient's email address.

The mail is only sent on monday's,

To test for current day change the 'if' condition
eg, For Wednesday -  "if day == 2:"
